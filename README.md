# editorconfig-cli

## Install

```bash
pip install editorconfig-cli
```

## Usage

```
usage: editorconfig-cli [-h] [--exclude [EXCLUDE ...]] [--use-gitignore] path

Format files based on .editorconfig

positional arguments:
  path                  Path to the directory to format

options:
  -h, --help            show this help message and exit
  --exclude [EXCLUDE ...]
                        Files or directories to exclude
  --use-gitignore       Use .gitignore to exclude files
```
